# Raylib-Template
 super easy to use starting point with raylib
 ## To use
- open main.code-workspace with VScode
- $`git remote rm origin`  
- $`git remote add origin...`
- Terminal -> build
- $`./main`

## pre-reqs
- install VS Code 
- add C/C++ extention
- install raylib with MinGW compiler
- download this repo
